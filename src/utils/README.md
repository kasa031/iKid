# Utils Directory

Dette mappen inneholder hjelpefunksjoner og utilities som brukes gjennom applikasjonen.

## Filer

- `validation.ts` - Valideringsfunksjoner (e-post, passord, telefon, etc.)
- `helpers.ts` - Generelle hjelpefunksjoner (formatering, beregninger)
- `messaging.ts` - E-post og meldingsfunksjonalitet
- `exportLogs.ts` - Eksport av logger til CSV
- `testDataGenerator.ts` - Generering av fiktive testdata
- `notifications.ts` - Notifikasjonshåndtering for kalenderoppføringer
- `optimization.ts` - Ytelsesoptimalisering (debounce, throttle, memoize)
- `colorContrast.ts` - WCAG-kontrastberegninger
