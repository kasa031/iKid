# Changelog

Alle betydelige endringer i iKid-prosjektet vil bli dokumentert i denne filen.

## [1.0.0] - 2025-11-07

### Lagt til
- Grunnleggende prosjektstruktur med React Native og TypeScript
- Firebase Authentication og Firestore database
- Innloggings- og registreringsskjermer
- Inn- og utkryssing for foreldre og ansatte
- Oversiktsskjerm med søk og filtrering
- Barn-profiler med visning og redigering
- Kalender med hendelser
- Logger for inn/utkryssinger
- Innstillinger med språkvelger og dark/light mode
- Admin-funksjoner (legg til barn, ansatte, knytt foreldre)
- Flerspråklig støtte (norsk, engelsk, polsk)
- E-postfunksjonalitet for ansatte
- Rollebasert tilgangskontroll
- GDPR-compliant datalagring
- Dokumentasjon (README, brukermanual, sikkerhet, GDPR)

### Sikkerhet
- Passord hashing via Firebase Authentication
- Rollebasert tilgangskontroll
- Firestore Security Rules
- Kryptert datatransport

- README med installasjonsinstruksjoner
- Database-dokumentasjon
- Sikkerhetsdokumentasjon
- GDPR-dokumentasjon
- Brukermanual

- Firebase setup-guide
